🔧 QUẢN LÝ MYSQL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Start MySQL:
  brew services start mysql

Stop MySQL:
  brew services stop mysql

Restart MySQL:
  brew services restart mysql

Check Status:
  brew services list | grep mysql

Connect to MySQL:
  mysql -u root

Show Databases:
  mysql -u root -e "SHOW DATABASES;"

Hiện tại MySQL KHÔNG CÓ PASSWORD (development mode)

Nếu muốn set password cho user root:
  mysql_secure_installation

Hoặc:
  mysql -u root -e "ALTER USER 'root'@'localhost' IDENTIFIED BY 'your_password';"

Sau khi set password, nhớ update trong wp-config.php!

❌ "Can't connect to MySQL"
  → brew services restart mysql
  → Đợi 5 giây rồi thử lại

❌ "Access denied for user 'root'"
  → Xác nhận Password field để trống
  → Hoặc dùng: mysql -u root (test trước)

❌ "Database already exists"
  → Không sao! Duplicator sẽ dùng database đó
  → Hoặc xóa và tạo mới:
    mysql -u root -e "DROP DATABASE dranmytas_local;"
    mysql -u root -e "CREATE DATABASE dranmytas_local;"
