❌ "Database already exists"
   → Chưa chọn đúng Action dropdown
   → Phải chọn "Connect and Remove All Data"

❌ "Can't connect to database"
   → Check MySQL running: ./check_mysql.sh
   → Hoặc restart: brew services restart mysql

❌ "Access denied"
   → Password để TRỐNG (không điền gì)
   → Username phải là: root

❌ "Unknown database"
   → Database name gõ sai
   → Phải là: dranmytas_local (không có space)
